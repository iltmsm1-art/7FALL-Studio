# 7FALL™

**Shake the Can. Reveal Your Fate.**

7FALL™ is an original premium push-your-luck tabletop tile game created by Anthony Mays.

Players shake a distinctive black-and-gold canister, draw seven wooden tiles, and decide whether to accept their Fate Row or risk up to three discards in pursuit of a higher-scoring combination.

## Features

- Premium tabletop game experience
- Fast 10–20 minute gameplay
- 1–4 Players
- Ages 8+
- Physical and digital editions
- Commercial product currently in development
